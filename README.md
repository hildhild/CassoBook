### CassoBook

## Set up
1. Chuyển vào thư mục web_source
```cmd
cd web_source
```
2. Install các module (và nên sử dụng node các phiên bản 16.0 || 18.0 || 20.0)
```cmd
npm i 
```
## Debug
1. Chuyển thư mục root thành web_source. 
```bash
npm run dev
```
2. Kết nối với ứng dụng qua đường dẫn
```url
	https://localhost:3000
```
